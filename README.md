# 🫆 Mitt portfolio

Här hittar nu mitt portfolio där kravlistan för uppgiften såg ut som följande:

  

1. En bild med tillhörande korrekta attribut inkl. alt-text
2. En navigation/meny (räcker med att den leder till sektioner på samma sida, behöver ej vara separata sidor)
3. En "huvudsektion" med själva innehållet på sidan (dvs. < main >)
4. Tre "projekt" dessa kan vara dummies/placeholders om du inte har något nu.
5. Ett sidofält med information (< aside >)
6. rubtiker på minst 3 nivåer, dsv. H1, H2, H3
7. En länk till en extern webbplats
8. Sidan ska funger på både mobil, tablet och desktop (dvs. ska skala "sömlöst" mellan de olika upplösningarna)
9. Ett kontaktformulär som innehåller "kontaktsökarens" namn, e-post, webbplats, telefonnummer, den dropdown, en checkbox och ett textfält fr paragraftext (textarea) och en skicka-knapp. Formuläret ska inte gå att skicka iväg.
10. Kontaktuppgifter till dig
11. En sidfot
